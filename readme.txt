Embed Shopnetic Widget into your WordPress site.

== Description ==

Shopnetic Widget plugin is the easiest way to embed widget videos in WordPress.

= Features =

* Pretty Shopnetic widget embed
* Responsive and mobile friendly

== Installation ==

1. Go to the Add New plugins screen in your WordPress Dashboard
1. Click the upload tab
1. Browse for the plugin file (shopnetic.zip) on your computer
1. Click "Install Now" and then hit the activate button
